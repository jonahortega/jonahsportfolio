## Quick Instructions:

1. Visit one of the links above
2. Download the model in **GLB** or **GLTF** format
3. Save it as `hand.glb` (or `hand.gltf`) in this directory: `public/models/`
4. The app will automatically load and style it with cyberpunk theme!

**Note:** If no model file is found, the app will use a procedural fallback hand.

## Which one to choose?

- **Best for realism**: Option 1 (CGTrader Free Hand)
- **Best for animation**: Option 2 (CGTrader Rigged Hand)
- **Best for commercial use**: Option 3 (Sketchfab - CC Attribution)
